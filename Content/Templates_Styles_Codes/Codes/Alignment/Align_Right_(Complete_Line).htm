Align - Right (Complete Line)

(See also: Align Right (Single word)) - (Old Code: [cd...])

Keystroke: Shift + F7

Code Used: [fr] + [:] + [;]

What does it do?

It forces the word, or group of characters, immediately following the command to the right hand margin. This command is cancelled by a space or carriage return. It is not necessary to have any characters before the command.

Where would it be used?

To force a single word or string of characters to the right of the page as shown below.

Unit 6
270 Littleton Road
Westford

Usage in DBT:

[fr][:]Unit 6[;]
[fr][:]270 Littleton Road[;]
[fr]Westford

NOTE: In the above example only the [fr] code is required on the third line, because there is only one word.

Produces in Braille:

                      ,unit #f

         #bgj ,littleton ,road

                        ,we/=d
