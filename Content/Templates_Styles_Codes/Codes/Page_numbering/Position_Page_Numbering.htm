Position Page Numbering

(See also: Document: Page Numbering)

Code Used: [svpnpN:N:N:N]

Keystroke:

What does it do?

Sets the position and occurrence of both braille and print page numbering.

This code is not needed if the page number positions are fixed throughout a document, because the page numbering can then be set using Document: Page Numbering for editing page number positions.

Where would it be used?

When the default needs to be varied or changed to suit local requirements.

Usage in DBT:

The Ns above are in the following order

1st) Braille page number on odd sides.

2nd) Braille page number on even sides.

3rd) Print page number on odd sides.

4th) Print page number on even sides.

Values of N can be as follows:

0 = No number.

1 = Upper left.

2 = Upper right.

3 = Lower left.

4 = Lower right.

Produces in Braille:

The default for American Textbook format is [svpnp4:4:2:2] I.E. Braille page numbers on lower right of page, and Print Page numbers on upper right.
