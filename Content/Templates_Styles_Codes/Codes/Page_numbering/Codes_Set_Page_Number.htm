Set Page Number on New Page

(See also: Set Page Number and New Page)

Code Used: [pgN]

Keystroke: Available from Layout Menu.

What does it do?

It forces a new page, and sets the starting page number of the new page.

Where would it be used?

Wherever you wish to start a new page and change the page number.

Usage in DBT:

End of Table of Contents
[pg1]
This is a new story.

Produces in Braille:

,end of ,table of ,contents

------------ page break ------------

,this is a new story4       #a

Let us explain!

The pg code forces a new page. N is whatever page number you wish to begin with.
