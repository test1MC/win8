Set Number and Prefix on New Page

(See also: Page Headers & Footers)

Code Used: [pgN~X]

>Keystroke: (None at present)

What does it do?

It forces a new page immediately after the code, and sets both Page Number and Prefix

Where would it be used?

Perhaps where a book ends and appendices follow.

Let us explain!

The pg code forces a new page at this point. 1 is the number we wish to begin with, and the A is the Prefix to the Page Number.
