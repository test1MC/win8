Arabic v Roman

Code Used: [pnta] and/or [pntr]

Keystroke: Select Page Numbering from Document menu.

What does it do?

These are two very simple Codes which will change the page number format between Arabic and Roman numbers when the number next appears.

Where would it be used?

To change page numbering from Arabic to Roman and vice versa.

Usage in DBT:

[pv1][pntr]This is the Preface of our book.

[pg1][pnta]This is Chapter one of the book.

Let us explain!

The [pv1] code at the beginning of the document sets the page number to one without creating a new page, with the [pntr] setting the Style of number to Roman.

[pg1] starts a new page with the number one, while [pnta] sets the Style to Arabic numbers.
