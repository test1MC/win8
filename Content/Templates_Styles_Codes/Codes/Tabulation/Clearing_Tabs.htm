Clearing Tabs

Before you can set Tabs, you need to know how to clear them

What you need to know first: What are Tabs?

See Also: Setting Tabs

Code Used: [ctb]

Keystroke: (None at present)

What does it do?:

It removes all Tab settings.

Where would it be used?

At the point from which you wish Tabs to be removed, or reset.

Usage in DBT:

[ctb]

At the point to wish Tabs to be cleared. Usually one or more [stb] codes follow after [ctb].
