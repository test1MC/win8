Embosser Character Table

In some Codes, for example where you require Leader or Guide dots, you need to specify what braille character(s) you wish. The DBT code requires that you enter a character. (E.G. If you want Dots 3 and 6, a Dash or Hyphen is required.)

The following table shows what braille characters are normally embossed when DBT's braille file is sent to an embosser.

You may need to refer to this table for some of the Topics in the Codes section in order to determine what character you need to complete a code.
