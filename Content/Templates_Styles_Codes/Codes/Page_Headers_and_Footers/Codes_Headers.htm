Headers

See also: Footers

Code Used: [svrhpN] [tls] [tle] [tld] (Old Codes: [tlc] [tlp])

Keystroke: Can be accessed from Document Menu

What does it do?

It places a Header on pages from the next page after which the Code is inserted. However, if placed at the very beginning of a document, the code will apply to the first page also.

Where would it be used?

Wherever in a document you wish to commence a running Header.

Usage in DBT:

[svprhp3]
[tls]Summary of Requirements[tle]
Bla bla bla bla bla bla bla[tld]

Produces in Braille:

    summ>y ( ,require;ts    #a

,bla bla bla bla bla bla bla

Let us explain!

The[svprhpN] code is used to determine whether the Header is to appear on odd pages only, even pages only, or both odd and even.

N can be, 1 for odd pages only, 2 for even pages only and 3 for both odd and even pages.

[tls] begins the Header and [tle] ends it.

Note that [tld] can be used to stop headers appearing from that point forward.
