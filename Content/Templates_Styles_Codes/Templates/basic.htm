Basic Templates

The Duxbury Braille Translator produces braille for more than 140 world languages. For a good number of those languages, DBTincludes a customized template that recognizes the local formatting rules. For most languages of the world, DBT offers a basic template.

A basic template has a rich collection of styles to meet many needs. Each basic template is the same except for the default braille translation table. The collection of styles in a basic templates is designed to meet the typical user's most common requirements, for example:

  • Importing files from Microsoft Word (either as doc or docx) - the basic templates map almost all built-in Word styles into useful styles in DBT
  • Importing files using the Daisy/NISO tag set (devised in 2005) - the basic templates map these tags into DBTstyles
  • Importing HTML files - again, the basic templates map these tags into useful DBTstyles

In addition, the collection of styles in a basic template handles many of the typical forms of character emphasis (bold, italics, underlining), some math-related styles from the math importer, and some specific DBT styles needed for braille tasks not found in the inkprint world (such as computer braille).

You will find that some of the Daisy/NISO styles have no effect in DBT. There was an attempt to have a DBT style for each of the tags in the Daisy/NISO system, but quite a few of these tags have no impact on braille formatting.

No matter whether the source of a document is Word, HTML, or some other source, using a basic template should produce translated braille in a reasonable format without much extra manipulation.

However, for those who want much finer control over the braille format similar to that in the customized templates, and for those who wish for a more sophisticated approach to styles, click here to learn about creating your own custom DBT Template for importing Word documents, but adapted for your language of choice.

Basic DBTCharacter Styles: Emphasis and Language

To use these styles in DBT, highlight the text, press F8, and pick one of these styles:

  • Biblical-langs -- handle the text as Greek/Hebrew/Syrillic, etc.
  • bold -- emphasis
  • English_UEB1 -- handle the text as grade 1 Unified English Braille
  • English_UEB2 -- handle the text as grade 2 Unified English Braille
  • English_UK_old -- handle the text as grade 2 British Braille (prior to Unified English Braille)
  • French -- handle the text as grade 1 Unified French
  • German -- handle the text as grade 1.5 German (a very small number of simple contractions)
  • IPA -- handle the text as International Phonetics Alphabet
  • Italian -- handle the text as grade 1 Italian
  • italic -- emphasis
  • Portuguese -- handle the text as grade 1 Portuguese
  • Spanish -- handle the text as grade 1 Spanish
  • Russian -- handle the text as grade 1 Russian
  • underlined -- emphasis

To help locate the right choice more quickly, you can press the first letter of the DBT style right after pressing F8. For example, to style a sentence as Russian, highlight the sentence, press F8, then R, then press the down arrow until you can select Russian, by pressing <Enter>.

The IPA translator is an international braille standard. All of the other languages are translated the way they are done in the dominant country. For example, Spanish produces Spanish braille as it is done in Spain, French as it is done in France, etc. We recommend that you experiment with the language styles to make sure you understand what they do.

You can also create your own styles to make use of other braille translators or contraction modes. These styles are the ones included in all the Basic templates.
