ALBANIAN - Technical

SPECIAL REQUIREMENTS AND LIMITATIONS

Although DBT is able to display accented letter combinations, it is nevertheless better to use Microsoft Word or Open Office for entering and editing print text, which can then be imported into DBT for subsequent translation.

When preparing the text in Word (or Open Office), be sure to use a Unicode font (such as Lucida Sans or Times Roman), so that the underlying characters are encoded in Unicode. (Note that the appearance on screen is not the issue. Fonts that merely cause standard ASCII characters to be displayed as the desired accented letters will not work, because they will be imported according to their standard interpretation, not their appearance.) To recognize a non-Unicode font, open a file in Word, highlight a sentence and apply the font Times Roman. If the sentence is no longer readable, you have been using a non-Unicode font.
