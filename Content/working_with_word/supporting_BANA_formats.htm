Supporting BANA Formats, 2011 (United States and Canada)

Getting Oriented

Supporting BANA’s Braille Formats Principles of Print-to-Braille Transcription, 2011, is a complex subject. This formatting is used by the United States and Canada.

Much of the effort to support BANA Formats has centered on the Microsoft Word templates created by Susan Christensen. See Using the BANA Braille 2014 Template for more information.

The basic premise is to format your Word document using the BANA Braille 2014 template. Open the Word file in DBT , using the English (American Textbook DE) - BANA template. Or use the English (American Textbook DE) – BANA Nemeth template for Nemeth. This procedure supports many of the formatting guidelines from BANA Formats, 2011.

Questions about the support of BANA Formats 2011 can be addressed to languages@duxsys.com
