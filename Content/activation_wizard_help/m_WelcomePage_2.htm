Welcome, Licensed User, Not Yet Activated

Your Duxbury product is licensed but is not activated. This means you have received your Duxbury product license number but have not started the Activation process.

You need to select one of the options for activation from the Main page. If you are a single user, select "Activate this computer".

If you are one of several users in a multi-user network, you probably need to select the "Connect" task. Check this with your IT Support people.

If you are the person setting up this computer to be the license server for your network, you need to select "Set up this server to host the network license".
