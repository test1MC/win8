Term definition for: CanDoInterpoint

This embosser is capable of doing interpoint (two sided) embossing. If you want to. you can give a command to your embosser to only emboss on a single side. If you do so. you can note that in the embosser setup "device settings" by unclicking "Embosser os set up for interpoint:. If you want to. you could ask that a particluar document is embossed single sided in the Document / Embosser settings.
