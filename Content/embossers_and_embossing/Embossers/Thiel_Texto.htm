Thiel Texto

Current Vendor information about the Thiel Texto

The Thiel Interpoint is a high-performance interpoint embosser that cannot do graphics. It only has a serial interface. To change the parameters, you need to run a program from your computer. The English language setup program is called SETX10UP.EXE. Set the unit for MIT code (this is their term for North American character set).
