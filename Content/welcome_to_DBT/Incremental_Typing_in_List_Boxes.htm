Incremental Typing in List Boxes

A List Box allows you to select from an alphabetical and/or numerical list of items contained within it. However, if you are using a keyboard, once you are focused in the list, you can usually type in the first letter, and be automatically moved to the items beginning with that letter. In the example illustrated below, if we were to type the letter h, we would be moved immediately from acronym, to h1.

DBT now incorporates, "Incremental Typing", which allows you to type in more than one letter in order to narrow down the search within a letter group.

For example, our list might contain main items as below:

fame
far
foam
fore
forget
foxed
foxing

If we typed f, we would land on the first word, "fame". If we then typed o, we would jump forward to the first of the words beginning with "fo". If we then typed x, we would jump forward again to words beginning with. "fox".

If you make a mistake, or wish to start with another letter, pressing backspace will allow you to re-enter from scratch. (i.e. you cannot backspace to correct letters after the first)
