Keyboard Shortcuts for DBT on the Macintosh

A keyboard shortcut, also known as an accelerator, allows you to trigger an action without stepping through the menus with the normal menu keys. Listed below are DBT's shortcut keys:

Function Keys
Windows Macintosh Description
F1 [no shortcut] Open the Help viewer.
F2 F2 View menu, Six-Key Entry
F3 ⌘ S File menu, Save As
F4 F4 Layout menu, Left Margin, 3
F5 F5 Layout menu, Code List
F6 ⌘ R Edit menu, Replace
F8 F8 Layout menu, Styles
F9 * Insert Code
F10 * Activate Menu Bar
F11 F11 Edit menu, Find Misspelling
Alt + F2 ⌥ F2 Layout menu, Character Codes, Hard Space
Alt + F3 ⌥ F3 View menu, Codes
Alt + F4 ⌘ Q File menu, Exit
Alt + F7 ⌥ F7 Layout menu, Line Codes, Flush Right w/Leader
Ctrl + F4 ⌘ F4 File menu, Close
Ctrl + F5 ⌘ F5 View menu, Print Font (Braille File)
Ctrl + F6 ⌘ F6 View menu, Braille Font (Braille File)
Ctrl + F7 ⌘ F7 View menu, SimBraille Font (Braille File)
Ctrl + F8 ⌘ F8 View menu, Translated Line (Braille or Print File)
Ctrl + F9 fn F8> Insert Style
Ctrl + F11 ⌘ F11 Document menu, Embosser Setup
Ctrl + F12 ⌘ F12 Document menu, Printer Setup
Shift + F1 Missing Layout menu, Line Codes, Single Spacing
Shift + F2 ⇧ F2 Layout menu, Line Codes, Double Spacing
Shift + F3 ⇧ F3 ?
Shift + F4 ⇧ F4 Layout menu, Right Margin, 6
Shift + F6 ⇧ F6 Layout menu, Line Codes, Centering
Shift + F7 ⇧ F7 Layout menu, Line Codes, Flush Right
Shift + F8 ⇧ F8 Apply Last Style
Shift + F11 ⇧ F11 Edit menu, Spell Check (Print File)
( Checks from cursor downwards)
