Checking Out the User Interface

The Translated Line

In the DBT Editor, when you are in a braille file (*.dxb) you can see one line at the bottom of the page showing the inkprint of the line that contains the cursor. If it is not present, press Control-F8. The improvements in DBT 11.3 include many to make sure the translated line is a good translation of the braille line.

Default Embosser Output

When you use the Global, Embosser Settings to set up a New Embosser, one of the prompts on the General Settings Panel is Output Options. The choices are Print, Port, or File. The choice Port refers to a serial or parallel port. Because serial and parallel ports are no longer in common use, the default setting is now Printer (i.e., a Windows printer). We expect this simple change in the default value to cut down in the number of tech support calls. See this page for more help in setting up an embosser in DBT.

The Hyperlink Import Choices

Go to the Global, Import Options Menu. The Hyperlink Import Choices are on the top of this screen. The prompts are fairly clear about the consequences of the different settings.

Removing Spaces around Math Signs of Comparisons

Go to the Global, Import Options Menu. Find the checkbox labeled Removing Spaces around Math Signs of Comparisons. Do not check this box for the lower grades, but do check it for the upper grades. This is an issue for the United Kingdom. In the United States, Nemeth math is often used instead of UEB math, so this would not apply.

Removing Spaces around Dashes

Go to the Global, Import Options Menu. Find the checkbox labeled Removing Spaces around Dashes. Do not change this unless you are in Australia. If you are, we presume you know which setting to use.

Code Page Specification

When you Open a file in DBT (especially a textfile), there is a new prompt Code Page (Character Set) at the bottom of the file open choices. The choice of a Code Page is only important when importing a textfile. The ability to specify the Code Page is of importance to those in Central Europe. For most users, keep this at Windows 1250.
