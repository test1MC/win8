Use of the New Languages and Language Improvements

If you are familiar with producing braille in a particular language, then seeing the improvements can be as simple as re-translating a document again, to see if the braille is improved.

If you are not familiar with producing braille in a particular language, then seeing the improvements can be more difficult. You may need a sample file for a language that you do not use.

There is a link on the Duxbury web site [Requires an internet connection] which helps someone test different braille translation tables, even if brand new to DBT, and do not know the language being tested.
