Page Numbering Questions

One of the most common support subjects which arise relates to page numbering, and here we will try to explain in more detail.

Many page numbering problems are caused by the incorrect positioning of the required changes within the text.

With some of the questions here, you should know how to view and edit DBT's Codes. To reveal Codes, the keyboard command Alt-F3, will toggle codes on or off. See The Menus: View: Codes topic for more details.

Details about editing Codes is detailed in The Menus: Templates, Styles and Codes: Codes: Amending Codes.

The questions here are broken into two groups:

  1. Braille Page Numbering
  2. Print Page Numbering (Often referred to as "Reference Page Numbering")

Braille Page Numbering:

1. How do I start a file with a page number other then 1?
Ensure your cursor is at the very beginning of the document. From the Layout menu select "Set Page Number", or for a more detailed explanation of the Code used see "Codes: Set Page Number"

2. How do I change the page number without changing the page?
Ensure your cursor is at the very beginning of the page you wish to begin a new number sequence at. Then from the Layout menu, select "Set Page Number...".

3. How come after I change the page number its not working?
Ensure that your page number immediately follows the New Page Code. There must NOT be any other Codes between the New Page Code and the Page Number Code.

4. How come after I change the page number it is ignored until the next page?
Ensure that your page number immediately follows the New Page Code. There must NOT be any other Codes between the New Page Code and the Page Number Code.

5. How do I change to Roman page numbers?
First ensure that your cursor is in the correct position. The from the Layout menu, select "Set page number" ensuring that you check the "Roman" radio button. See more information at "Layout: Set Page Number..."

6. How to I change from Roman back to Arabic page numbers?
First ensure that your cursor is in the correct position. The from the Layout menu, select "Set page number" ensuring that you check the "Arabic" radio button. See more information at "Layout: Set Page Number...".

7. How do I ensure that the next page changes to an odd page? ( the [sd] code)
See Skip to Odd of Even.

8. How do I create prelim page numbers such as p1 or t1? ( the [sd] code)
See Set Page Number Prefix, or Set Number and Prefix on New Page.

Print (Reference) Page numbering:

1. How to I get print page numbers in a Duxbury document?
If you wish to manually add print page numbers into your DBT Text file, you will need to use the [lea] Code. Place your cursor the exact point where page number requires to be located, hold down Ctrl and press the left square bracket to bring up the Codes dialog. Between the square brackets which will appear, type "lea" (without the quotes) and then click OK. Now immediately after the Code, which will appear as [lea] in your file, enter the print page number followed by a space, then continue with the normal text of your document. See Print Page Numbers for more details.

2. How do I get print page numbers in a Duxbury document from a Word document?
If you wish to automatically import Word's print page numbers, go to DBT's Global menu, and select "Word Importer". Here you will find appropriate options in the "Print Page Breaks:" drop down list.
