Character Code Table - (U+04xx) - Cyrillic Script

Note: The following table may not display all characters in the Chr column on your system. This is entirely dependent on the version of Windows you are running, and what Character Sets are installed.

Support for Cyrillic characters is found on Windows XP and all subsequent editions of Windows.

Many of the DBT translation tables support Cyrillic characters. These Include Belarusian, Bosnian, Kirghiz, Macedonian, Mongolian, Russian, Serbian, Tajik, Ukrainian, and Uzbek.
