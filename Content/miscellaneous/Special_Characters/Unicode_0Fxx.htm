Character Code Table - (U+0Fxx) - Tibetan

The Tibetan script is used in two languages supported by DBT : Tibetan and Dzongkha.

Note: The following table may not display all characters in the Char column on your system. This is entirely dependent on the version of Windows you are running, and what Character Sets are installed. These fonts are built into Windows 7 and Windows Vista. For Tibetan script, we recommend the use of Windows 7 or Windows Vista.

Text using these scripts can be imported from Microsoft Word into Duxbury DBT. If you use DBT or above then choose the Tibetan or Dzongkha translator. The Tibetan translator supports your choice of Tibetan script or Wylie Romanization.
