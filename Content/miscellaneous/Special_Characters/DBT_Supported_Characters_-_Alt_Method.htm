The Alt Key Entry Method in Windows and DOS

Although superseded by more convenient methods on most Windows systems, these "Alt-key" methods can still be useful in some circumstances, for certain characters. In the list below, note the number next to "DOS:" (if available) or "Win:", according to the system in use, and follow these directions:

MS-DOS users: Hold down the Alt key while entering, on the numeric pad, the three digits shown as the DOS value; then release the Alt key. In some cases, you may need to have "Num Lock" on to do this.

Windows users: You may use the Alt-nnn method given above for DOS users, or if you prefer you may use a similar Alt-key approach but with the 4 digits given as the Win value (always starting with zero).
