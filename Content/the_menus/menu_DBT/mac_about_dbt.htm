About DBT

Keystroke: None; select from the DBT menu.

Windows users may notice that on the Macintosh platform "About DBT" appears in a different location from the Windows platform.

There are two very important items on this screen which you will require if you contact Duxbury or your Duxbury dealer for any reason. The Version number and date which may follow, and your license number.

If you are using a screen reader, simply pressing the tab key will allow you to hear the Version and License Number.

Please Note: The example license number "DBT K1234-5678" is purely for illustrative purposes.
