Layout: Footer...

Keystroke: f from Layout menu.

Keystroke: Select from Layout menu

The Footer dialog is used to create or change the running FooterFooter. There are two controls.

The first control is a text box for the text of the Footer. If you are editing a braille document, the text box should contain braille. If six-key editing is enabled, chordingchording is used to enter text, otherwise you would type in ASCII braille equivalents. If you are working on a print document, the text box must contain print; the contents will be translated to braille along with the rest of the document.

The second determines where the Footer will appear. You can choose 'No Footer' if you wish to discontinue an earlier Footer.
