Layout: Page Codes: New Reference Page

Keystroke: Alt + Enter or R from Page Codes menu.

Keystroke: ⌥↩

New reference page will produce a CodeCode to show where the page changed in the source document. Normally this is used to mark the print pages when creating a braille textbook. After using this command, you should type the number of the new reference page, followed by a space. Leader DotsLeader Dots will appear, with the reference page flush right against the margin.
