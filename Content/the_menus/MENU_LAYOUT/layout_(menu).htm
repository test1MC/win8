LAYOUT: (Menu)

Keystroke: Alt + l (l as in Lima)

The Layout menu is used to handle all aspects of translation and formatting of text within your document. In other words, Layout is where you can access the numerous StylesStyles, translation Codes Codes,and format codes that make DBT so flexible. Whenever possible, and especially with major projects, we recommend using Styles instead of direct coding. Doing so will make it easier to manage your documents. For an in-depth discussion, we recommend reading Codes & Styles - Introduction.
