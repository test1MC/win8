Layout: Translation Codes: Grade 0

Keystroke: Alt + 0 (Zero)

Keystroke: ⌥0 (Zero)

This menu selection will cause DBT to insert a series of codes which, when the document is translated, will cause the Translator to render text following in Grade 0Grade 0 braille.

Advanced users may wish to customize which codes are inserted by using "Global: Shortcut Preferences..." on page 1Global: Shortcut Preferences.
