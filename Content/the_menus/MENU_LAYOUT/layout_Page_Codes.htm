Layout: Page Codes

Keystroke: g from Layout menu

Keystroke: Select from menu

The page codes pull out listpull-out list contains several choices that can be used to insert codes which control pagination. Included within Layout: Page Codes are:

New Page (Ctrl + Enter)(⌘↩)

New Reference Page (Alt + Enter)(⌥↩)

Block Protect (There is no keyboard shortcut)

Vertical Centering (There is no keyboard shortcut)
