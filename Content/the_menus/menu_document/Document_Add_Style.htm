Document: Add Style...

Keystroke: a from Document menu

Keystroke: Select from menu

You can use this menu feature to create your own Styles.Styles.

When you select Document: Add Style..., you will be presented with the Dialog Boxdialog box shown above, in which you should enter a name for the new style. If you do not, the following warning will be displayed.

Unless your new Style is to apply to an entire paragraph, check the box marked "Style can be nested within another style."

You may also select an existing style on which to base the new one.

You may now enter Beginning Codes and Ending Codes as required.

See also: Document:Modify Style...
