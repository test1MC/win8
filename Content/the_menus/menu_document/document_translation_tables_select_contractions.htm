Document: Translation Tables: Select Contractions...

Note: This item appears at the end of the Translation Tables menu.

For fuller details of the contents of the following, please see the Miscellaneous: Graduated Tables topic.

This selection is currently available for "English/American" translation or" English/British (new)", as selected using Document: Translation Tables:.

Choosing this menu item calls up the WeeBee Dots dialog box, through which you can choose a set of contractions in steps progressing from grade 1 to grade 2 braille.

"English/American" gives a choice of the NLS I.M.B.T. course, or the TSBVI Cluster course.

"English/British (New)" is for the UK Fingerprint course, currently supplied by the R.N.I.B.

These options progress from using only a few contractions to using a full set of grade 2 braille contractions. The contractions included with each selection are listed in the Learning topic.
