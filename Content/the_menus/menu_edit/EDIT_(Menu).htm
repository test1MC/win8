EDIT: (Menu)

Keystroke: Alt + e

Keystroke: Select from menu.

This menu and its sub-menus deal with editing work to be done within a file.

(Some items may be grayed out depending on when you call up this menu.)

This menu contains the following Items:

"Undo" on page 1 Undo (orCtrl + Z⌘Z)
"Redo" on page 1 Redo (or Ctrl + Y⌘Y)
"Cut" on page 1 Cut (or Ctrl + X⌘X)
"Copy" on page 1 Copy (or Ctrl + C⌘C)
"Paste" on page 1 Paste (or Ctrl + V⌘V)
"Clear" on page 1 Clear (or Del)
"Select All" on page 1 Select All (or Ctrl + A⌘A)

"Edit: Highlight" on page 1 Highlight (or Ctrl +H)

"Change Case" on page 1 Change Case (or Shift + F3⇧F3)

"Insert File" on page 1 Insert File... (There is no keyboard shortcut. This command opens a dialog)

"Goto..." on page 1 Goto (or Ctrl + G⌘G)
"Find..." on page 1 Find (or Ctrl + F⌘F)
"Find Again" on page 1 Find Again (or Ctrl + I⌘I) (I as in India)
"Replace..." on page 1 Replace (or F6⌘R)

"Spell Check..." on page 1 Spell Check (or Shift + F11⇧F11)
"Find Misspelling" on page 1 Find Misspelling (or F11)
