Help: Translate Help

Keystroke: t from Help menu.

Keystroke: Select from menu

When you select this option, you will be directed to a web site where you can generate a web-based translation of DBTHelp in over 90 languages. Select the language of your choice. (Click on the appropriate flag.)

You will see a similar structure to the Help system you are viewing now, and you will be able to browse the system in your chosen language.

Please Note: First, you must have an Internet connection for this feature to work. Second, the selected language is translated automatically by a web-based computer program, and therefore it cannot be assumed to be 100% accurate.
