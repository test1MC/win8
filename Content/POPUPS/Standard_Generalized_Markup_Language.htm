Standard Generalized Markup Language (SGML) is a specification for ASCII text files that allows all text in a document to be marked according to the purpose it serves in the document. For example, in an SGML file, a heading might be written as follows:

<heading>Chapter 3</heading>

In this form, not only is the text of the heading ('Chapter 3') obvious, but so too is the fact that this text represents a heading.

SGML facilitates sharing documents among different products that may offer different methods for viewing or printing the document.
