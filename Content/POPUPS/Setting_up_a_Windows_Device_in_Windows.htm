Setting up a Windows Device (printer driver) to send data to your embosser is essentially the same as setting up any printer driver. Follow these steps:

  1. From the Desktop, open My Computer.
  2. Open Control Panel.
  3. Open Printers.
  4. Open Add Printer.
  5. Follow the Add Printer Wizard to install this printer driver. When prompted for the manufacturer and model of your printer, select 'Generic' as the manufacturer, and 'Generic/Text Only' as the model.
  6. Give this printer driver a name that describes your embosser. Do not make it the default printer for Windows programs.
  7. It is usually advisable not to print a test sheet to your embosser.
