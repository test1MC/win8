An acronym for American Standard Code for Information Interchange, ASCII is a generic text-based file format in which each character is represented by a number. For example, 'a' is 97, 'b' is 98 and 'c' is 99. Because it is such a widely known standard, almost all programs can import and export ASCII text, including programs on other computers, such as MacIntosh VAX and Unix.

DBT can import and export several 'flavors' of ASCII.

All document files created by versions of DBT before 10.1 are ASCII text.
