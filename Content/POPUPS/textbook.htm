There are special rules used for the production of textbooks in braille. The current print page number needs to be readily available to the braille reader. In DBT, print page numbers in a braille document are referred to as reference page numbers. Textbook format is easily created by using the correct template when first creating your document.

The essential elements are that page numbers go in the lower right and reference page numbers in the upper right. This can be accomplished through the Document: Page Numbering... dialog if your template did not handle this already.

To indicate where the reference page turns, use Layout: Page Codes to select New Reference Page. Type the number of the upcoming page (with number sign if in a braille document) and space bar to move on.
