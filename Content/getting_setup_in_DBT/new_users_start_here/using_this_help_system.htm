Using This Help system

This help system consists of five parts:

The Contents tab works like a traditional table of contents. Topics are grouped by subject. Just double-click on a topic to view the topic in the right-hand frame.

The Index tab allows you to search for a specific subject. Any topics with that index entry will appear, and you can select which one you want to view.

The Search tab allows you to add topics, so you can come back to them at any time.

The Favorites tab allows you to add Topics which you may want a quick way of coming back to.

The Glossary tab provides a list of short definitions of words and/or terms which you may find useful.

Across the top is a series of Icons. From left to right they are:

Hide - which will hide the Table of Contents, thus widening the topic pane.

Back - takes you to the previously viewed Topic.

Forward - only active if you have used Back, and takes you forward again.

Home - Returns you to the beginning Help screen

Print - to print the Topic in view.

Options - provides a few customizing options.

DBT Web Site - Opens up Duxbury's web site in the Topic pane.

Emboss - Allows you to emboss the Topic in view.

Screen reader users, and others interested in keyboard navigation of the online help can find detailed instructions here: Navigating Help.
