Setting up your printer.

On the Computer:

If you have a printer (for ordinary print, not for braille), it should be set up in the normal way as a Windows printer. This is usually done through the "Add Printer" facility within "Printers" in the Control Panel, or equivalently in Start > Settings > Printers, choosing the driver that matches the printer as supplied by Windows or the printer manufacturer.

If you have a printer (for ordinary print, not for braille), it should be set up the same way as for any printer on your Mac. This is normally done through the System Preferences: Print & Scan dialogue.
