Please press right arrow to open this book. Press down arrow, Enter, and then F6, to move to the Screen Reader Navigation Topic for advice on moving around the DBT Help system.

If you have a wheel mouse, hold down the Control key and scroll the wheel to adjust the size of font displayed on the screen. Alternatively, you may set the font size via Internet Explorer's "View" menu, then "Text Size".
