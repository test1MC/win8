External Browser Links

Many web sites are unhappy when opened in Flare.

The solution is to set Flare to open the link in the browser as opposed to within Flare/Help.

This simply requires selecting the "Target Frame" as "New Window/Tab" when Inserting or Editing a Hyperlink as shown in the following screen shot.

A further benefit in some cases is that users can then close Help and work in DBT while the web page remains open. e.g. to copy and paste.
