Introduction

SWIFT (Send Word file Immediately For Translation) was originally developed by a U.K. DBT user many years ago to help in fine tuning some aspects of Word files prior to importing them into DBT. It became very popular as there was more and more demand for importing Word files.

But with this demand came requests for additional features from users themselves. One in particular was to map Word Styles to DBT braille Styles, especially where Custom Word Styles were being used.

As time went on, users wanted a facility to make a selection in Word, and only import that into DBT. Now you can select text which is not even contiguous. As DBT's math braille capability increased, many users wanted to simply import the likes of MathType equations.

One of the new visitors to the duck pond, Tim Burgess from the UK, was tasked with implementing many users' suggestions, and in the specifications for doing this was a line which read, "Make George happy!".

Well, I am happy, and I hope you will be too.
