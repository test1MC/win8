Introduction

This Quick Start Guide is intended to assist you with the following tasks:

  1. Installing Perky Duck,
  2. Starting Perky Duck,
  3. Creating a document in Perky Duck,
  4. Saving the document,
  5. Sending the document to your instructor, and
  6. Finding out more about using Perky Duck.

Installing Perky Duck

Usually, Perky Duck is installed by running a program on the Duxbury Systems web site. To do this, you need to use a web browser program connected to the internet. These instructions assume that you are using Internet Explorer, which comes with Microsoft® Windows™. If you prefer to use another browser, you will have to adapt the instructions somewhat.
