Keyboard Shortcuts.

A keyboard shortcut, also known as an accelerator, allows you to trigger an action without stepping through the menus with the normal menu keys.  Listed below are Perky Duck's shortcut keys:

Function Keys.

F1 = The Help Key

F3 = File, Save As

F10 = Activate Menu Bar

Alt + F4 = File, Exit

Ctrl + F4 = File, Close

Ctrl + F5 = View, Print Font

Ctrl + F6 = View, Braille Font

Ctrl + F7 = View, SimBraille Font

Ctrl + F11 = Document, Embosser Setup

Ctrl + F12 = Document, Printer Setup

Character keys

Alt + D = Document menu

Alt + E = Edit menu

Alt + F = File menu

Alt + G = Global menu

Alt + H = Help menu

Alt + V = View menu

Ctrl + A = Edit, Select All

Ctrl + C = Edit, Copy

Ctrl + E = File, Emboss

Ctrl + G = Edit, Goto

Ctrl + H = Edit, Highlight

Ctrl + N = File, New

Ctrl + O = File, Open

Ctrl + P = File, Print

Ctrl + S = File, Save

Ctrl + V = Edit, Paste

Ctrl + X = Edit, Cut

Ctrl + Y = Redo

Ctrl + Z = Undo

 

Other keys:

Shift + Arrow keys = Highlight text

Ctrl +Shift + Down Arrow = Select from cursor to end of paragraph
