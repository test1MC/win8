File: Save.

Keystroke: Ctrl + s

Please try to remember this keystroke and use it regularly.

(Please also see details of the Autosave Options feature in the Global menu.)

This command tells Perky Duck to save your document into a disk file. If your document already has been saved, then any changes are written immediately. Otherwise, you will be prompted for a file name and location, as shown below.

(This Save dialog box works in exactly the same way as a Windows Save As dialog box.)

The above is the standard Windows Save as: dialog where normal Windows conventions apply.

File Name: Perky Duck may automatically supply a file name. You may change the file name. If Perky Duck has not automatically supplied a file name, then you must type one. YOU DO NOT need to type the file extension. You may use the drop down menu to choose whether to save the document as a Duxbury Braille (.dxb) file or a formatted braille file (.brf). Your braille instructor will tell you which he or she prefers.
